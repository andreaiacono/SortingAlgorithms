Insertion Sort

works by repeatedly taking the next item and inserting it into the final data structure in its proper order with respect to items already inserted. 

Source code:

for (i = 1; i < values.length; i++) {
	j = i;
	int iValue = values[i];
	while ((j > 0) && (values[j - 1] > iValue)) {
		values[j] = values[j - 1];
		j--;
	}

	values[j] = iValue;
}
